# Abdul AI 🤖

ระบบจัดการคลังสินค้าอัจฉริยะ (AI-Powered Inventory Assistant)

## วิธีการใช้งาน

1. เปิดไฟล์ `.env` และใส่ `GEMINI_API_KEY` ของคุณ
2. เข้าใช้งานผ่านเบราวเซอร์ที่ `http://localhost/Abdul%20AI/`
3. พิมพ์คำถามภาษาไทย เช่น "สินค้าที่เหลือน้อยกว่า 10 ชิ้นมีอะไรบ้าง" หรือ "มีอะไรที่กำลังจะหมดอายุในเดือนนี้"

## โครงสร้างโปรเจค

- `index.php`: หน้าจอสนทนาด่วน (Turbo Mode - Intent Engine)
- `api/abdul-chat.php`: API สำหรับ Turbo Mode (ประมวลผลทันทีใน < 100ms)
- `api/chat.php`: API ตัวเดิมสำหรับใช้ LLM (รองรับคำถามแบบซับซ้อน)
- `api/export.php`: ส่งออกข้อมูลเป็นไฟล์ Excel
- `includes/functions.php`: ฟังก์ชันหลักและระบบความปลอดภัย
- `logs/query_log.txt`: บันทึกการใช้งานและ SQL ที่เกิดขึ้น

## ความปลอดภัย

- ระบบอนุญาตเฉพาะคำสั่ง `SELECT` เท่านั้น
- มีการ Log ทุก Query ที่ส่งมา
- แนะนำให้ใช้ DB User ที่สิทธิ์ Read-only เท่านั้น
