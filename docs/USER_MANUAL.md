# User Manual

## Login

1. Open the Visual Location web app
2. Enter username and password
3. Handheld scanners use the same login (shorter session)

## Dashboard

Shows material counts, near-expiry items, MQTT / Raspberry / Socket.IO status, and recent stock movements.

## Search

Scan or type HanaPart / PUID → view location → **Highlight** to show on TV rack and physical IO lights.

## Receive material

**Receive Reservation**: scan PUID, select slot, confirm quantity.

**Receive Return**: scan returned PUID, select slot, confirm quantity.

## Picklist

Open picklists from CPK, issue PUIDs, close when complete.

## Rack overview

Live rack map; updates via Socket.IO when inventory changes.

## Expiry check

Filter materials approaching expiration.

## Reports

Export current stock, movements, receive history, pick history, and expiry reports to **Excel**, **CSV**, or **PDF**.

## TV display

Open `/tv` on rack monitors (kiosk key if configured).

## Language

Switch TH / EN from user profile or admin language settings.
